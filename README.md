# 💰 BudgetBuddy – Expense Tracker

A modern, responsive, and beginner-friendly expense tracker web application built using HTML, Bootstrap 5, and JavaScript (ES Modules). It helps users track income, expenses, and overall financial balance in real time.

---

## 📂 Repository

The full source code for BudgetBuddy is available on GitHub.  
This project follows a modular ES6 architecture for clean separation of concerns.

## ✨ Features

- Add income and expense transactions
- Edit and delete transactions
- Search transactions by title
- Filter by category, type, and date range
- Real-time balance, income & expense summary
- Interactive Chart.js visualization
- LocalStorage data persistence
- Light/Dark mode toggle
- Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES Modules)
- Chart.js
- LocalStorage API

## 🧠 Architecture

- app.js → Entry point (controls app flow)
- transactions.js → CRUD operations
- storage.js → LocalStorage handling
- ui.js → DOM rendering & UI updates
- filters.js → Filtering logic
- calculations.js → Financial calculations
- utils.js → Helper functions
